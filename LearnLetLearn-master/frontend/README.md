# L2 Platform Frontend

## Setup Instructions

1. Install dependencies:
   npm install

2. Start development server:
   npm start

## Project Structure
- src/
  - pages/
    - Login.jsx
    - Register.jsx
    - Profile.jsx
    - Skills.jsx
    - Match.jsx
    - Chat.jsx
    - Video.jsx
    - Requests.jsx
  - components/
    - Navbar.jsx
    - Sidebar.jsx
    - ChatBox.jsx
    - VideoCall.jsx
    - RequestList.jsx
  - hooks/
  - utils/

## Features
- React.js SPA
- Tailwind CSS for styling
- React Router for navigation
- Axios for API calls
- Socket.io-client for real-time chat/video
- Modular structure for scalability
